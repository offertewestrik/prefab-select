# Photos for La Vigor

The site shows branded placeholders wherever a photo file is missing, so it
looks complete before any real photography is added. Drop real photos at the
exact paths below and they replace the placeholders automatically — no code
changes needed.

> **Permission first.** Only use photos you own or have permission to use
> (the café's own Google Maps / Instagram / Facebook photos). Don't pass
> stock photos off as La Vigor.

## Tips
- Use `.jpg` and keep each file under ~400 KB for fast loading.
- Landscape for hero/spotlight, square for menu items, mixed for gallery.
- Shoot in warm, natural light to match the brand mood.
