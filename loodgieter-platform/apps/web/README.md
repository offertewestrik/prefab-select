# Loodgieterplatform — web (Next.js)

De publieke marketingsite en het vakman-dashboard.

- Offerte-/factuursysteem met per-vakman huisstijl (logo, bedrijfsgegevens,
  accentkleur + balkkleur, IBAN, voettekst) op de PDF.
- Dashboardmodules: klanten, planning, werkbonnen, facturen, offerte-follow-up,
  AI-bedrijfscoach, reviews, onboarding, eigen prijzen en monteurmodus.

Draait op Vercel (project `loodgieterplatform0412`); alleen dat project deployt
deze repo (zie `apps/web/vercel.json`).
